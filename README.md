# TaskFlowAPI
ASP.NET Core Web API Task Management Project using EF Core and SQL Server


ASP.NET Core Web API project for managing tasks.

## Features
- User Authentication
- Create Task
- Update Task
- Delete Task
- Task Status Management
- Entity Framework Core
- SQL Server Database

## Technologies Used
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication
